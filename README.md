## 簡介
  你好, 這個專案係我係ERB前端網站開發人員入面的一份個人功課
## 功課要求
  * 設計一個旅遊資訊網站(前端),該網站最少包含5個頁面。
  * 網站需介紹四個旅遊地點,每個地點來自不同的國家。網站應包括有關地點的歷史、文化和景點等訊息,以及為旅行者提供的實用資料。
  * 網站同時包含以下基本內容:網站標題和標誌,圖片、文字、導覽列、包含連結至其他頁面和視頻等多媒體元素。
  * 該網站應具有視覺吸引力、用戶友好性及互動性等元素。
